# Olden Era Templates

Repositorio donde guardo mis plantillas personalizadas de *Heroes of Might and Magic: Olden Era* y los assets auxiliares necesarios para construirlas y distribuirlas.

## Estructura del repositorio

- [`Templates/`](./Templates/): plantillas jugables y sus assets asociados

## Plantillas incluidas

Por ahora el repositorio incluye:

- [`Ways Around`](./Templates/Ways%20Around/): plantilla para 2 jugadores en modo `Single Hero`, centrada en flanqueos exteriores, presión por ruptura central y batallas de torneo programadas

Cada carpeta de plantilla puede incluir:

- el fichero principal `.rmg.json`
- imágenes de vista previa y diagrama
- zips de assets o ficheros fuente auxiliares usados para construirla
- instrucciones de instalación específicas de la plantilla

## Propósito

Este repositorio cumple dos funciones:

1. Servir como punto de distribución para plantillas personalizadas terminadas.
2. Servir como espacio versionado para iterar sobre esas plantillas con el tiempo.

## Notas

- El repositorio está organizado pensando primero en GitHub, no como una carpeta lista para copiar directamente al juego.
- Las carpetas de plantilla incluyen instrucciones de instalación indicando qué hay que copiar dentro de `StreamingAssets/`.
